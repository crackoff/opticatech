О проекте

Данный сервис разрабатывается одним человеком в свободное время, и преследует следующие цели:

  • Внести свой вклад в популяризацию технологий дополненной реальности в сфере интернет-торговли;
  • Сдедать чуточку более комфортными покупки в интернет-магазинах;
  • Глубже изучить технологии обработки изображений.

Для того, чтобы снизить риски для пользователей данного сервиса, автор принял решение сделать свободными его исходные коды.

Сервис доступен бесплатно всем желающим, однако Вы можете сделать пожертвование на его развитие.

На что пойдут деньги:

  • Покупка сертификата SSL;
  • Повышение тарифного плана хостинга для улучшения производительности;
  • Оплата труда разработчика.

Ближайшие цели в развитии сервиса:

  • Разработка личного кабинета;
  • Разработка автоматического распознавания очков на фото при загрузке их в личный кабинет;
  • Вход при помощи Yandex, Google, vk;
  • Создание профилей в социальных сетях для размещения информации об обновлениях;
  • Улучшение основного алгоритма;
  • Разработка алгоритма для работы с 3D-моделями.
